[题目传送门](https://www.luogu.com.cn/problem/P11229)

<!-- more -->

首先直观的想法就是 $n$ 比较大的时候，尽可能多放 8，然后在 $n$ 比较小的时候分类讨论即可。

或者看数据范围提示就是根据 $n \mod 7$ 的值进行讨论。

但是注意一个细节，当 $n=10$ 时答案是 22，但 $n=17$ 时答案应该是 200 而不是 228。这也是本题唯一坑点。

还是附上代码吧。

```cpp
int main()
{
	a[6]=0,a[2]=1,a[5]=2,a[4]=4,a[3]=7,a[7]=8;
	t=read();
	while(t--)
	{
		n=read();
		if(n<=1)
			cout<<-1;
		else if(n==6)
			cout<<6;
		else if(n<=7)
			cout<<a[n];
		else 
		{
			if(n%7==1)
				cout<<10,n-=8;
			else if(n%7==3)
				if(n==10)
					cout<<22,n-=10;
				else
					cout<<200,n-=17;
			else if(n%7==4)
				cout<<20,n-=11;
			else if(n%7==6)
				cout<<6,n-=6;
			else if(n%7)
				cout<<a[n%7],n-=n%7;
			for(ri i=1;i<=n/7;i++)
				cout<<8;	
		}
		cout<<"\n"; 
	}
	back 0;
}
```
