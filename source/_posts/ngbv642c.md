$20+10+0+0=30$，机房内并列 $Rank$ $2$。

由于版权原因放不了题面。简单略写一下四道题的题意吧。

T1：给定一个字符串，只能由 ```
							aaaaa
							``` 和 ```
		bbb
	``` 和数码构成，数码必须在末尾且不能有前导零，子串中 ```
	aaaaa
	``` 至少要出现一次。定义一个字符串的价值为 ```
	aaaaa
	``` 子串出现的次数 $×a+$ ```
	bbb ```   子串后出现的  ```
    	aaaaa``` 子串数之和 $×b$。相同价值的字符串，字典序大的价值高。求长度为 $n$ 的价值第 $k$ 小的字符串，无解输出 $-1$。
    
T1 题解：

看到 $1≤k≤10^9$，发现对于 $n≥15$ 的情况都是 ```
	aaaaa```加上后面一堆数字。于是可以把这部分特判掉。对于 $n≤14$ 的情况，发现排列组合情况很少，直接分类讨论即可。
 
 
T2：给出一个 $5×4$ 的标准华容道棋盘，给出各个棋子的位置（$1×2$ 的棋子不保证横竖），定义一步为棋子移动一个单位，求曹操逃离的最小步数，无解输出 $-1$。

T2 题解：

显然可以大力 bfs。但是代码不太好写。考虑到每个格子只有五种状态（空，被 $1×1$ 占据，被 $2×1$ 占据，被 $1×2$ 占据，被 $2×2$ 占据），可以使用状态 hash，状压一下之后发现不同的状态并不多，直接做就可以。

代码实现的细节是移动滑块的时候可以先把滑块挪出棋盘，再加到新的位置，这样比较方便，且能避免很多问题。

T3：平面上有 $n$ 个点，每个点向上下左右移动 $1$ 步的代价为 $1$，，定义一个位置集联通当且仅当该位置集中任意两个位置都可以只经过该位置集中的点互相到达，求 $n$ 个点位置两两不同且位置集联通的最小代价。

 T3 题解：
 
 这题非常恶心的限制是 $n$ 个点位置两两不同。假如没有这个限制，那么聚集点显然是（所有点横坐标的中位数，所有点纵坐标的中位数），这个结论大概可以随便猜出来。
 
 考虑加上题目限制怎么做。不难发现，最后聚集完毕后出现了一个大连通块。考虑数据范围 $1≤n≤6$，可以直接爆搜最后形成的连通块形状，然后再枚举每个人走到哪个位置（显然离自己近的位置比较优），距离就是曼哈顿距离。
 
 注意这题要开 $long$ $long$。

T4：有 $n$ 个人，一人依次说一句话，可能是真话或者假话。他们说话可能有三种形式：1、只有 $k$ 个人说的是真话（包括自己）。2、下一个人说的是真话。3、下一个人说的是假话。根据这些信息判断每个人所说话的真假，有多组解则输出任意一组合法解，无解输出 $-1$。

T4 题解：

考虑如果没有第一种话，那么我们可以很清楚地知道谁说了真话，谁说了假话，只需要从第一个人一直向下推即可。

有了这个想法，我们自然可以想到分段处理，对只有第二种话和第三种话的段进行预处理。

那么问题就是怎么处理第一种话的情况。显然我们必须知道这个人说的话是真话还是假话，不然没法做。那么我们不妨枚举有几个人说了真话，这样就可以知道每个说第一种话的人的说话真假，然后其余的人就直接处理即可，最后判断是否合法即可，如果没有一种情况合法，则无解。
