<!-- more -->

CSP-S 2020 T2。摸鱼摸到现在终于开始补题了……

这题考点不知道是啥，简单位运算？感觉比较偏阅读理解，变量和题意都很乱……

简化版题意：有分别编号为 $0∼2^k-1$ 的 $2^k$ 种动物，动物园里现在有 $n$ 种动物。有 $m$ 个要求，第 $i$ 个要求使得若有某种动物的编号用二进制表示，第 $p_i$ 位为 $1$，则需要购买第 $q_i$ 种饲料。求根据购买的这些饲料，还能再养多少只动物。

题目中 $q_i$ 保证互不相同，因为我们考虑能够饲养的动物必然是 $p_i$ 位为 $1$ 的动物，所以我们只关心 $p_i$ 的限制，因此 $q_i$ 是什么并没有用。于是显然 $c$ 也没有用。（给那么多无用变量干扰思路是吧 SB 出题人）

然后考虑先处理出是否有某种动物满足 $p_i$ 位 $=1$ 的限制。这个部分很好处理，对于每一个 $a_i$，直接让它与 $1$ 进行按位与，然后每次右移一位即可，复杂度 $O(nk)$。

下记 $res[i]=1$ 代表存在第 $i$ 位为 $1$ 的动物。

考虑我们不能加入任何 $p_i$ 位为 $0$ 的动物，因此如果 $res[p_i]=0$， 那么意味着有一半的动物都不能养了，于是相当于动物总数变为 $2^{k-1}$。（因为动物编号的某一位非 $0$ 即 $1$，少了一位就相当于否定了一半的情况，且一定不会影响到现在动物园里已养的动物）。

不妨设出现 $res[p_i]=0$ 这种情况的数量为 $s$，则答案就是 $2^{k-s}-n$。

细节1：由于 $k \le 64$，要特判一下 $k=64,n=0$ 的情况。

细节2：输入 $p_i$ 的时候要记得判重，重复的 $p_i$ 只算作一次。

时间复杂度 $O(nk+m)$。

代码：
```cpp
#include<bits/stdc++.h>
#define ll long long
#define back return
#define ri register int
#define ull unsigned ll
using namespace std;
ll read()
{
	ll x=0,f=1;
	char ch=getchar();
	while(ch<'0'||ch>'9')
	{
		if(ch=='-')
			f=-1;
		ch=getchar();
	}
	while(ch>='0'&&ch<='9')
	{
		x=x*10+ch-'0';
		ch=getchar();
	}
	back x*f;
}
int sum,tot[105];
ull n,m,c,k,a,p,q,rec[105];//rec[i]=1 代表存在某种动物编号的第 i 位是 1 
int main()
{
	n=read(),m=read(),c=read(),k=read();
	for(ri i=1;i<=n;i++)
	{
		a=read();
		sum=0;
		while(a)
		{
			if(a&1)
				rec[sum]=1;
			a>>=1;
			sum++;
		}
	}	
	for(ri i=1;i<=m;i++)
	{
		p=read(),q=read();
		if(!rec[p]&&!tot[p])
			k--,tot[p]=1;
	}
	if(k==64&&n==0)
		cout<<"18446744073709551616"<<"\n"; 
	else
		cout<<(ull)pow(2,k)-n<<"\n";	
	back 0;
}
```
